    Models and training

    Models and training

    EmpiricalRiskMinimization.Model — Type.

    Model(...)

    The Model() function constructs an ERM model. The typical invocation is Model(U, V, Loss(), Reg()), where U and V specify raw inputs and targets, respectively, and Loss() specifies some type of training loss (default: SquareLoss()) and Reg() specifies some type of regularizer (default: L2Reg(1.0)). For more details, see the description of ERM models in the usage notes.
